How to write latex formulas with mathjax

Before writing formulas, several steps are required:

  1. Uninstall hexo-math: npm uninstall hexo-math
  2. Install: npm install hexo-filter-mathjax --save
  3. Add following configs to your _config.yml
  4. Uninstall hexo-renderer-marked: npm uninstall hexo-renderer-marked
  5. Install hexo-renderer-kramed: npm install hexo-renderer-kramed --save
  6. Change config of math.mathjax.mhchem to true
